# @koppelia/koppeliajs — game SDK

The SvelteKit SDK every Koppelia game installs (WebSocket, shared state, BLE hardware, UI). Published privately on GitHub Packages.

## Run

```bash
npm install   # in a game; needs @koppelia npm auth
```
